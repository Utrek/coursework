Метод get_commons_params() формирует параметры запроса.
Метод get_photos() класса VKAPIclient формирует json файл c данными всех фотографий профиля клиента
Метод photo_url_dict() формирует json файл с данными о сохранённых фото и словарь photo_dict сожержащий названия файлов и их url в ВК.
Метод create_folder() класса YaDiskClient создаёт папку на Яндекс диске
Метод upload_photo(self, url, name) загружает файлы из photo_dict в папку на Яндекс Диске.
Функция save_photo(id_client,token) создаёт объекты классов VKAPIclient,YaDiskClient и пользуясь методами классов создаёт json файл c данными всех фотографий профиля клиента, создаёт папку на Яндекс диске и загружает в неё фото.
